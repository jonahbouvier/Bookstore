Book Library App

The Book Library App is a simple Python application using Tkinter that allows users to manage a catalog of books and create their own library collection.

Features

- Book Catalog: View a list of books with their titles, authors, and genres.
- Add to Library: Mark books from the catalog as part of your personal library.
- My Library: View and manage your own library collection.
